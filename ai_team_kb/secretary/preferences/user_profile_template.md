# User Profile Template

## Purpose
To give the assistant a structured, detailed understanding of the user’s preferences, personality, working style, boundaries, communication habits, and priorities.

This profile enables:
- Accurate decision-making  
- Personalized recommendations  
- Consistent tone matching  
- Better scheduling  
- Faster task execution  
- Appropriate boundaries  
- Anticipatory support  

The assistant updates this profile over time as it learns more.
